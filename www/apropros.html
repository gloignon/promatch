À propos de l'application promatch

Promatch est une application web permettant d'accéder aux fonctionnalités de base de la librairie MatchIt pour le langage statistique R. Le but est de simplifier l'utilisation de techniques d'appariement statistique afin de les rendre accessibles aux non-spécialistes. L'appli a été créée par le Prof. Guillaume Loignon de l'Université du Québec à Montréal.

Qu'est-ce que MatchIt?
MatchIt est une librairie pour le langage statistique R créée par Daniel Ho et ses collègues. MatchIt sert au prétraitement des données en vue d'une inférence causale par le biais de méthodes d'appariement statistique. Il améliore la qualité de l'inférence en réduisant l'influence indésirable de covaraibles. Il existe un article de 2011 décrivant le package en détail, voir aussi les recommandations décrites dans un article de Ho et al. (2017), les deux articles sont indiqués dans la Bibliographie ci-dessous.

Aide

Comment utiliser promatch ?

  1. Sélectionnez le fichier contenant votre jeu de données. Cliquez sur Parcourir, puis sélectionnez le fichier .csv ou .xlsx sur votre ordinateur. Le fichier sera téléchargé.
  2. Sélectionnez vos variables. Après avoir sélectionné votre fichier, des champs vont apparaitre pour vous permettre de sélectionner la variable de traitement (regroupement), les covariables, et les variables d'appariement exact (facultatives). La variable de traitement doit être une variable binaire (dichotomique) ; c'est elle qui divise vos données en deux groupes pour la comparaison. Les covariables sont des variables que vous soupçonnez ou savez avoir une influence sur le résultat, mais qui ne sont pas le principal objet de l'étude. Cela pourrait inclure des facteurs démographiques comme l'âge et le sexe, ou d'autres variables de fond. Dans le contexte de l'appariement, les covariables sont utilisées pour ajuster les groupes afin qu'ils soient comparables sur ces caractéristiques de fond. Les Variables d'Appariement Exact sont optionnelles et utilisées lorsque vous souhaitez vérifier que les individus appariés soient soient identiques sur certaines caractéristiques clés. Lorsque vous utilisez l'appariement exact, l'algorithme va apparier des individus ayant exactement la même valeur pour ces variables.
  3. Sélectionner la méthode d'appariement.
  4. Sélectionner la mesure de distance.
  5. Click the Match button to start the matching process. A message will appear to confirm that it's working.
  6. Cliquez sur le bouton Télécharger le CSV apparié pour télécharger les données appariées résultantes. Ce bouton sera désactivé si les données d'appariement ne sont pas disponibles pour le téléchargement. Une boîte affiche des informations sur la qualité de l'appariement. Il s'agit d'une sortie directe de la fonction summary() de R appliquée à l'objet d'appariement.
