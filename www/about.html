About the promatch Shiny App

Promatch is a simple RShiny web app that allows you to access the basic functionalities of the powerful R package, MatchIt. This application is designed to simplify the process of statistical matching, making it accessible to both statisticians and non-statisticians alike. It was created by Prof. Guillaume Loignon from University of Quebec in Montreal.

What is MatchIt?
MatchIt is a comprehensive package for the R statistical language, created by Daniel Ho and colleagues. It is used for pre-processing data for causal inference through statistical matching methods. It improves the quality of inference by reducing model dependence and bias in estimation. There is a 2011 paper describing the package in details, see also the recommendations described in an article by Ho et al. (2017), both are linked in the Bibliography below.

Help!

How does it work?

  1. Select your dataset. Click Browse, then select the .csv or .xlsx file on your computer. The file will upload.
  2. Select your variables. Fields will appear to let you select the treatment (grouping, or pairing) variable, covariates, and optional exact matching variables. The treatment variable needs to be a binary (dichotomous) variable; it is what divides your data into two groups for comparison. The covariates are variables that you suspect or know have an influence on the outcome but are not the primary focus of the study. These might include demographic factors like age and gender, or other background variables. In the context of matching, covariates are used to adjust the groups so that they are comparable on these background characteristics. The Exact Matching Variables are optional and used when you want to ensure that matched units are identical on certain key characteristics. When you use exact matching, the algorithm will only match units that have the exact same value on these variables.
  3. Select the Matching Method
  4. Select the Distance Measure
  5. Click the Match button to start the matching process. A message will appear to confirm that it's working.
  6. Click the Download Matched CSV to download the resulting paired data. This button will be disabled unless there is actual pairing data available for download. A box will display information about the quality of the matching, this is a direct output from R's summary() function applied to the matching object.
